# Hutool Module Map

- Source root: `D:\projects\hutool-v5-master`
- Core utils: `hutool-core`
- HTTP client: `hutool-http`
- Crypto: `hutool-crypto`
- Cron support: `hutool-cron`
- Configuration support: `hutool-setting`

For jimuqu-agent, Hutool is most useful for:

- file IO
- text and path helpers
- time and date formatting
- UUID and ID helpers
- lightweight HTTP helpers when Solon AI does not already own the call path
